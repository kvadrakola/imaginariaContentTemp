# República de Asteria — Sitio Oficial

Sitio web multipágina del proyecto académico **FrontEnd HTML5 & CSS3**.

## Cómo ver

1. Cloná o descargá este repo
2. Abrí `index.html` en tu navegador (doble click o `python -m http.server`)
3. No hace falta instalación ni dependencias — todo es HTML y CSS puro

## Páginas

- **index** — portal + navegación
- **identidad** — bandera, escudo, himno, lema, idioma
- **geografia** — relieve, hidrografía, mapa, clima, demografía
- **gobierno** — sistema político, constitución, derechos
- **economia** — moneda Lumen, pilares económicos, tecnología
- **cultura** — Cocido Asteriano, Día del Alba
- **historia** — mito fundacional, línea temporal
